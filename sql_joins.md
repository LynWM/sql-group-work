## SQL JOINs

- The **JOIN** clause is used to combine two or more tables, based on a related column between them.

### Different types of SQL JOINs

    1. INNER JOIN - returns rows that have matching values in both tables.
    2. LEFT (OUTER) JOIN - returns all rows from the left table, and only the matched rows from the right table.
    3. RIGHT (OUTER) JOIN - returns all rows from the right table and only matched rows from the left table.
    4. FULL (OUTER) JOIN - returns all rows where there is a match in either left or right table. NULL is used to represent the missing vlues if there are no matching rows in one of the tables.
    5. CROSS JOIN - combines every row from the left table with every row from the right.
